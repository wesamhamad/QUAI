# 🎬 نظام توليد الفيديو من النص

نظام بسيط لتوليد مقاطع فيديو من النصوص باستخدام AI، مصمم خصيصاً للعمل على Apple Silicon (M4).

## 📋 المتطلبات

- Python 3.9+
- 16GB RAM (متوفر لديك ✅)
- Apple Silicon M4 (متوفر لديك ✅)
- اتصال بالإنترنت لتحميل النموذج في المرة الأولى (~7GB)

## 🚀 الاستخدام

### 1️⃣ الاستخدام الأساسي

```bash
python3 generate_video.py "وصف الفيديو المطلوب"
```

### 2️⃣ أمثلة

```bash
# مثال 1: قطة تلعب
python3 generate_video.py "A cute cat playing with a ball in a sunny garden"

# مثال 2: منظر طبيعي
python3 generate_video.py "A beautiful sunset over the ocean with waves"

# مثال 3: مدينة
python3 generate_video.py "A busy city street with cars and people walking"
```

## ⚙️ الإعدادات

يمكنك تعديل الإعدادات في ملف `generate_video.py`:

- **num_frames**: عدد الإطارات (الافتراضي: 16)
  - كلما زاد العدد، كان الفيديو أطول ولكن يستهلك ذاكرة أكثر

- **num_inference_steps**: خطوات التوليد (الافتراضي: 25)
  - كلما زاد العدد، كانت الجودة أفضل ولكن أبطأ

- **height/width**: الدقة (الافتراضي: 256×256)
  - يمكن زيادتها إلى 512×512 لجودة أفضل

## 📝 ملاحظات مهمة

### التحميل الأولي
- في المرة الأولى، سيتم تحميل النموذج (~7GB)
- يتم حفظ النموذج محلياً في `~/.cache/huggingface/`
- بعد ذلك لن تحتاج للتحميل مرة أخرى

### الأداء
- توليد فيديو 16 إطار يستغرق 5-10 دقائق على M4
- الذاكرة المستخدمة: ~8-12GB
- كلما قللت الدقة وعدد الإطارات، كان أسرع

### الجودة
- النموذج المستخدم: `text-to-video-ms-1.7b`
- مناسب للأجهزة المتوسطة
- جودة جيدة لكن ليست احترافية (للاحترافية تحتاج HunyuanVideo على GPU قوي)

## 🔧 استكشاف الأخطاء

### خطأ: Out of Memory
**الحل**: قلل عدد الإطارات أو الدقة

```python
num_frames=8  # بدلاً من 16
height=128, width=128  # بدلاً من 256
```

### خطأ: Model not found
**الحل**: تأكد من اتصالك بالإنترنت في المرة الأولى

### بطء شديد
**الحل**:
1. قلل `num_inference_steps` إلى 15-20
2. قلل الدقة إلى 128×128
3. اغلق التطبيقات الأخرى لتوفير الذاكرة
